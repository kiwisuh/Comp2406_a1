To open the app:
1. Execute node app.js on command prompt.
2. Go to a web browser (preferably Chrome) and type: localhost:3000/index.html
3. Type in Either Brown Eyed Girl, Sister Golden Hair, or Peaceful Easy Feeling

Features:
1. Reads in file and output with chords on top and lyrics on bottom
2. Able to output song with chords using "Update Song" 
3. Able to move the lyrics and chords around the canvas

OS used: Windows 8
